# Spaltenmapping und Schemainferenz

In dieser Präsentation geht es um das automatische Zuordnen von Sensorsignalen aus Kraftwerken zu ihren entsprechenden Kennungen. Ferner wird ein Ansatz zur Erzeugung eines konzeptuellen Schemas aus solchen Kraftwerksdaten vorgestellt, das auf Grundlage der Informationstheorie und algebraischen Topologie mittels Verfahren des maschinellen Lernens geschätzt wird.

**Autor**: Luciano Melodia, Richard Lenz
